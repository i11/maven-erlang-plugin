erlang:edoc

Note:This goal should be used as a Maven report.

Full name:

eu.lindenbaum:maven-erlang-plugin:1.0.0-SNAPSHOT:edoc

Description:

Generates an EDoc report.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Binds by default to the lifecycle phase: generate-sources.

Required Parameters

Name Type Since Description
docOutput1 File - Directory to generate the EDoc report into.
Default value is: ${project.reporting.outputDirectory}/doc.
docOutput2 File - Additional directory to copy the generated EDoc into.
Default value is: ${project.build.directory}/doc.
srcMainErlang File - Directory where the sources reside.

Optional Parameters

Name Type Since Description
eDocOptions String[] - Additional options for EDoc.

Parameter Details

docOutput1:

Directory to generate the EDoc report into.
  • Type: java.io.File
  • Required: Yes
  • Default: ${project.reporting.outputDirectory}/doc

docOutput2:

Additional directory to copy the generated EDoc into.
  • Type: java.io.File
  • Required: Yes
  • Default: ${project.build.directory}/doc

eDocOptions:

Additional options for EDoc.
  • Type: java.lang.String[]
  • Required: No

srcMainErlang:

Directory where the sources reside.
  • Type: java.io.File
  • Required: Yes
  • Expression: ${basedir}/src/main/erlang/
